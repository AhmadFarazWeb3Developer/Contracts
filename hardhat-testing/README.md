# Testing Smart Contracts with Hardhat

This Project is testing my ERC20, Voting and Token Marketplace contracts

-> Installation Steps :

1. Create an empty folder
2. cd to created folder
3. npm init -y
4. npm install --save-dev hardhat
5. npx hardhat init
6. npx hardhat compile

-> Must Aware Be Aware Of:

1. Mocha: Unit testing framework
2. Chai: Assserting library
